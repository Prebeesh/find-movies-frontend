Personal Project Works

Find Movies

FindMovies is a platform designed to help users check the availability of their favorite movies and shows across various streaming services. The web application references all the major streaming platforms and includes movie details from sources such as IMDb and TMDb. The system's backend is powered by a set of MicroServices built using Spring Boot, while the frontend is developed using JavaScript, jQuery, and Bootstrap CSS. In the current release, FindMovies version 1.0, users can search for movies by entering the movie title and selecting the relevant country to check its availability.

Future Enhancements (FindMovies Version 2.0) - In the upcoming FindMovies version 2.0, we are introducing a new feature that will allow users to provide reviews for movies or shows available on the platform. The MicroService that will enable this feature is currently under development at FindMovies.

Sandesh

Sandesh is an Android application designed to improve the communication process within college advisory systems. The app allows student advisors to efficiently share important information with their students, a task traditionally managed through WhatsApp groups. Sandesh leverages an SMS gateway to send messages directly to students' or parents' cell phones, ensuring that critical information reaches them even when they are offline, reducing the chances of missed communication.

Developed by Prebeesh, Sandesh addressed a real-time challenge within his academic institution and was created during his college years. The project received recognition at the All Kerala IEDC Summit 2018.
