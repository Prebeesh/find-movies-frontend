Find Movies
Find Movies can be used for checking the availability of your favourite movies and shows over different movie streaming platforms. Different MicroService built in Spring Boot is supporting as the Backend for FindMovies. The Frontend of the system is built using Bootstrap. In FindMovies version-1.0, a user can input a movie name and appropriate country to check the availability of the movie. FindMovies version-2.0 will be adding a new feature where a user can give reviews to different movies or shows listed inside our application. The MicroService providing this extension is now in the workshop of FindMovies.

Sandesh
Sandesh is an Android Application developed for helping the college advisory system. The application helps student advisors in an academy to pass essential information to their students. Usually student advisors use WhatsApp groups to share those necessary information. Sandesh works with SMS gateway, so the information to the student/parent will reach them as SMS to their cell phones. And this approach reduces the chance for students/parents to miss the information even though they are offline.

This application developed by Prebeesh solved a realtime problem in his academy. And was done during his college days
